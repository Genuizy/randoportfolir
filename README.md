# Base44 Standalone Badge Script

An embeddable JS badge that displays the Base44 floating badge on web applications, tracks user actions, and offers clone/promo modals.

## Quick Start

Include the script tag before the closing `</body>` tag on your web application:

```html
<script 
  src="[https://your-cdn-domain.com/base44-badge.js](https://your-cdn-domain.com/base44-badge.js)" 
  data-app-id="YOUR_APP_ID"
  data-platform-url="[https://base44.com](https://base44.com)"
  data-allow-clone="true">
</script>
